# House Price Prediction using Machine Learning

This project predicts house prices using the King County housing dataset.

## 📊 Project Overview

The goal of this project is to build a machine learning model that predicts house prices based on property features such as:

- Bedrooms
- Bathrooms
- Square footage
- Floors
- Grade
- Location

The dataset used is the **King County House Sales dataset**.

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📈 Machine Learning Models

The following models were used:

- Linear Regression
- Ridge Regression
- Polynomial Regression Pipeline

## 🔍 Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Model Training
6. Model Evaluation
7. House Price Prediction

## 🎯 Result

The model predicts the estimated price of a house based on its features.

Example output:

Predicted House Price: **$326,772**
